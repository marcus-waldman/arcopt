# Initialize Flat-Ridge Detector State

Initialize Flat-Ridge Detector State

## Usage

``` r
init_flat_ridge_state(window = 10)
```

## Arguments

- window:

  Integer sliding-window length (default 10)

## Value

A list with empty diagnostic vectors and the window size.
